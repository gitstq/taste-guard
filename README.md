# TasteGuard - AI Content Quality Guardian

🛡️ A Python CLI tool that evaluates and optimizes AI-generated content quality,
helping users create unique, engaging, and non-generic outputs.

## Features

- 📊 **Content Quality Analysis** - Comprehensive metrics for text evaluation
- 🔍 **Generic Phrase Detection** - Identifies overused AI-generated expressions
- 📈 **Vocabulary Diversity** - Measures lexical richness and variety
- 💡 **Smart Suggestions** - Actionable recommendations for improvement
- 🎨 **Rich CLI Output** - Beautiful terminal formatting with colors

## Installation

```bash
pip install taste-guard
```

## Usage

```bash
# Analyze text
taste-guard analyze "Your text here..."

# Analyze from file
taste-guard analyze -f article.txt

# Compare two texts
taste-guard compare "Text A" "Text B"

# Get optimization suggestions
taste-guard optimize "Your draft text..."
```
